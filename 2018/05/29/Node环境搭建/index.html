Node 的安装

官网地址：https://nodejs.org
进入官网，根据自己系统平台选择下载

下载之后像安装软件一样，默认选项不用修改，一直下一步，最后安装完成。

Node安装好之后，可以在终端输入命令行查看版本检查是否安装成功。
